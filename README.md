Terminal iTunes Player
======================

A simple command line ruby based iTunes player.

Required Gems:
--------------

* [.appscript](http://appscript.sourceforge.net/) -- gem install rb-appscript

How To Run:
-----------

	./itunes.rb "<song-name>"
Ex:

	./itunes.rb "Afraid\ Of\ Everyone"
